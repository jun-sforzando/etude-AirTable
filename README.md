# etude-Airtable

## Requirements

- Python 3.8.5
  - python-dotenv 0.14.0
  - airtable-python-wrapper 0.15.0

## Setup

1. ターミナルで `python3 -m venv venv` を実行して仮想環境を作成
1. `source venv/bin/activate` を実行
1. `pip install -r requirements.txt` を実行
